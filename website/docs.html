SYSTEM PROTOCOLS

Welcome to the Cognito mainframe. Documented below are the standard operating procedures for local deployment.

INSTALLATION

Cognito operates on local silicon. Source compilation is required for initialization.

PREREQUISITES

Ensure the following dependencies are present in your environment:

  • Node.js (v18+): Download
  • Python (v3.10+): Download

QUICK_START

Execute the following command sequence:

1. Clone the repository:

git clone https://github.com/ArjunDeshwal/cognitoai.git
cd cognitoai

2. Initialize Python backend:

# Create virtual environment
python -m venv venv

# Activate (Windows)
venv\Scripts\activate

# Activate (Mac/Linux)
source venv/bin/activate

# Install dependencies
pip install -r backend/requirements.txt

3. Initialize Frontend & Launch:

cd app
npm install
npm run electron:dev
NOTE: Initial system boot may require additional time for backend service registration.

DEBUG

Boot Failure?

Verify Python venv activation status before executing npm run electron:dev. Check system PATH variables.

Latency Issues?

Confirm CUDA driver installation for NVIDIA hardware. Apple Silicon acceleration is enabled by default.
